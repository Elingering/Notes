## 通用命令
- keys [pattern] #遍历key
- dbsize #计算key的总数
- exists key #检查key是否存在 0 1
- del key [key...] #删除指定key-value 0 1
- expire key seconds #key在seconds秒后过期（也有其它单位的命令）
- ttl key #查看key的剩余过期时间 -1永不过期 0刚好过期 -2过期已删除
- persist key #去掉key的过期时间
- type key #返回key的类型
